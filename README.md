## How to Run

Since this is a plain HTML/CSS project, no installation or build step is needed.

### Option 1 — Open directly in browser

1. Clone or download this repository.
2. Navigate to the `Calculator` folder.
3. Double-click `calculatorNew.html` to open it in your default browser.

### Option 2 — Use Live Server (VS Code)

1. Open the `Calculator` folder in VS Code.
2. Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension if you haven't already.
3. Right-click `calculatorNew.html` and select **"Open with Live Server"**.
4. The app will open at `http://127.0.0.1:5501/Calculator/calculatorNew.html`.

## Features

- Clean gradient background (blue to purple)
- Supports basic arithmetic: addition, subtraction, multiplication, division
- Double-zero (`00`) and decimal (`.`) buttons
- Clear (`C`) and backspace (`<`) controls
- Highlighted `=` button on press

## Tech Stack

- HTML5
- CSS3
